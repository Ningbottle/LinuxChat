# 瓶子交流器 UI 设计规范

## 1. 设计概述

### 1.1 设计风格
- **整体风格**: 报纸复古 + 紫藤萝元素
- **设计语言**: 简洁、优雅、书卷气
- **灵感来源**: 
  - 登录页：报纸排版风格 + 地球仪元素
  - 聊天界面：宗璞《紫藤萝瀑布》风格的紫藤萝背景

### 1.2 设计目标
- 营造温馨、文艺的聊天氛围
- 保持界面简洁，突出内容
- 体现产品"瓶子交流器"的品牌个性

## 2. 字体规范

## 4. 布局规范

### 4.1 登录页
- **容器尺寸**: 440px × 560px
- **登录卡片**: 300px 宽
- **背景**: 报纸纹理 + SVG地球仪 + 装饰性文字片段
- **层级**: 卡片 z-index: 1，背景元素 z-index: 0

### 4.2 主聊天窗口
- **整体尺寸**: 960px × 640px
- **布局结构**:
  - 顶部栏：统一横跨整个宽度（包含Logo）
  - 侧边栏：240px 宽
  - 聊天区：flex: 1

### 4.3 侧边栏
- **用户列表**: 8px 内边距
- **用户项**: 10px 12px 内边距
- **当前会话**: 左侧2px深色边框 + 略深背景

### 4.4 聊天区
- **紫藤萝背景**: opacity: 0.35
- **消息区**: 20px 内边距
- **输入区**: 16px 20px 内边距

## 5. 组件规范

### 5.1 按钮
| 类型 | 圆角 | 内边距 | 边框 |
|------|------|--------|------|
| 主按钮 | 2px | 10px | 1px solid #44403c |
| 次按钮 | 2px | 10px | 1px solid #d6d3d1 |
| 连接按钮 | 2px | 9px 12px | 1px solid #d6d3d1 |
| 断开连接 | 2px | 6px 8px | 1px solid #e7e5e4 |

### 5.2 输入框
- **圆角**: 2px
- **内边距**: 10px 12px（登录）/ 10px 14px（聊天）
- **边框**: 1px solid #d6d3d1
- **Focus状态**: 边框变为 #78776c

### 5.3 消息气泡
- **圆角**: 2px
- **内边距**: 10px 14px
- **字号**: 15px
- **行高**: 1.6
- **发送消息**: 深色背景（#44403c）+ 白色文字
- **接收消息**: 白色背景 + 边框

### 5.4 头像
- **尺寸**: 28px × 28px
- **圆角**: 50%
- **背景**: #d6d3d1（默认）/ #78776c（在线）
- **字号**: 12px

## 6. 交互规范

### 6.1 过渡动画
- **标准过渡**: 0.15s ease
- **Hover效果**: 背景色变化、边框色变化

### 6.2 状态指示
- **当前会话**: 左侧边框 + 背景高亮 + 字体加粗
- **在线状态**: 绿色圆点（#10b981）+ 荧光效果
- **活跃会话**: 标题栏绿色圆点标识

## 7. SVG资源

### 7.1 地球仪（登录页背景）
- 线条风格：经纬线 + 椭圆
- 颜色：#78776c
- 透明度：0.07

### 7.2 紫藤萝（聊天背景）
- 图案尺寸：300px × 300px
- 元素：
  - 棕色主藤蔓（2-2.5px粗）
  - 淡紫色花穗（下垂瀑布状）
  - 绿色叶子
- 透明度：0.35

## 8. 响应式考虑

当前设计为固定尺寸（960×640），未来可考虑：
- 最小宽度：768px
- 侧边栏可折叠
- 消息区自适应高度

## 9. 实现注意事项

### 9.1 Qt/QSS映射
- `border-radius: 2px` → Qt中可能需要调整
- `backdrop-filter` → Qt不支持，需用其他方式实现
- SVG背景 → 可用QPixmap或自定义绘制

### 9.2 字体加载
- 需要将字体文件打包到应用资源中
- 优先使用系统已安装的字体
- 提供fallback字体链

### 9.3 性能优化
- SVG图案可预渲染为Pixmap
- 避免实时模糊效果
- 消息列表使用虚拟滚动
